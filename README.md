## How to run?
- remove example from example.env to be > .env
- remove example from examplesecrets.env to be > secrets.env
- populate the secrets with your creds and `sudo bash run.sh`

> [!note]
> - You must be sudo when run that script
> - GitHub password must be your token
> - Deploying to VM is not tested
